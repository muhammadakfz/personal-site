# muhammadakfz Personal Site

Personal portfolio and writing site for muhammadakfz, a software developer and physics student. The site combines a minimal portfolio homepage with longer notes about software, applied AI, physics, waves, and numerical ideas.

## Features

- responsive photo bento grid
- Markdown and MDX writing
- KaTeX math rendering with mobile-friendly equation overflow
- page and article transition animations
- contact page with an email form
- RSS feed, sitemap, robots.txt, and dynamic Open Graph images
- Vercel Analytics and Speed Insights
- light and dark color-scheme support

## Stack

- Next.js 16 and React 19
- TypeScript
- Tailwind CSS 4
- `next-mdx-remote`
- KaTeX, Remark Math, and Rehype KaTeX

## Development

```sh
npm install
npm run dev
```

Create an optimized production build with:

```sh
npm run build
```
